# Examples

Área dedicada para exemplos e templates do repositório.

## Estrutura

- `docs/prd`: exemplos de PRD
- `docs/rfc`: exemplos de RFC
- `docs/tdd`: exemplos de TDD por história
- `templates`: modelos reutilizáveis de documentos

## Observações

- Os exemplos atuais usam o caso de cartão margem como conteúdo.
- A organização da pasta é genérica para facilitar a entrada de novos exemplos no futuro.
